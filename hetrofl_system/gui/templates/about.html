Research Background & Methodology

Federated Learning Approach

Our system implements a novel heterogeneous federated learning approach that allows different types of machine learning models to collaborate without sharing raw data. This is particularly important for IoT network security where data privacy and distributed computation are critical.

Knowledge Distillation

We employ advanced knowledge distillation techniques to transfer knowledge from diverse local models (XGBoost, Random Forest, CatBoost) to a global neural network model. This enables the system to benefit from the strengths of different model architectures.

IoT Network Security Focus

The system is specifically designed for IoT network security applications, handling various types of network attacks including DDoS, injection attacks, malware, and other security threats commonly found in IoT environments.
